Weather App — веб-приложение для просмотра погоды по географическим координатам. Функционал: ввод широты и долготы, отображение текущей погоды (температура, скорость ветра, направление ветра), интерактивная карта местности через OpenStreetMap, различные иконки погоды в зависимости от условий, возможность добавления нескольких виджетов. Валидация координат: проверка числовых значений и диапазонов (широта -90 до 90, долгота -180 до 180). Используется Open-Meteo API для получения данных о погоде.
